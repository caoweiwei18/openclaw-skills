## 💡 新学习的知识

### 1. RustChain 平台特性
**关键发现**: RustChain 要求创建 Claim Issue 才会付款
- **流程**: PR → 验证 → **创建 Claim Issue** → 付款
- **时间**: 2-5 天
- **教训**: 不要等维护者联系，主动创建 Claim Issue

### 2. 百炼 API 特性
**关键发现**: 百炼提供 OpenAI 和 Anthropic 兼容端点
- **OpenAI 兼容**: `https://coding.dashscope.aliyuncs.com/v1`
- **Anthropic 兼容**: `https://coding.dashscope.aliyuncs.com/apps/anthropic`
- **配额管理**: 按月配额，需要监控使用情况

### 3. 敏感数据脱敏
**关键发现**: 在对话中暴露了完整的 API Key
- **格式**: `sk-sp-****...****` （只显示前缀和后 4 位）
- **存储**: `.env` 文件，不提交到 Git
- **行动**: 如果已泄露，立即撤销并创建新的

### 4. 结构化整理
**关键发现**: 定期整理可以大幅提升效率
- **频率**: 每周一次
- **内容**: 删除临时文件、创建索引、更新记忆
- **效果**: 工作区更整洁、查找更高效
